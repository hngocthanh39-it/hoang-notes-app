# 📝 Notes App

A simple browser-based notes application with local storage support.

## 🚀 Features
- Add notes
- Delete notes
- Data saved in browser (localStorage)
- Clean UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Use
1. Open `index.html`
2. Write a note
3. Click "Add Note"

## 🌐 Use Case
- Quick note taking
- Personal reminders
- Idea storage
